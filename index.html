Summary: Latitude vs. X

This project investigates if there is a correlation between latitude and weather patterns. It seeks to answer a fundamental question: What is the weather like as we approach the equator? Through successive weather data acquisitions, called API calls, we perform weather checks for 635 random cities all over the globe, but most specifically, at varying latitudes.
